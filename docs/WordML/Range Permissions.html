Range Permissions

Range permissions in a WordprocessingML document refer to a special type of bookmark used to control which subset(s) of users may edit a particular region of a document. Range permissions specify the user or set of users which are allowed to edit all content between them whenever the document protection specified by the documentProtection element (ยง2.15.1.28) is enabled and set to readOnly or comments.

Like bookmarks, range permissions are a legacy word processing function which predates the concepts of XML and well-formedness, so they can start and end at any location within a document's contents and therefore must use the "cross-structure" annotation format described in ยง2.13.2.

 </w:r>
 <w:permStart w:id="0" w:edGrp="everyone"/>
 <w:r>

 </w:r>

 <w:r>
 <w:t>.</w:t>
 </w:r>
</w:p>
