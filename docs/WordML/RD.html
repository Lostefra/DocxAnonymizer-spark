RD

Syntax:

RD field-argument [ switch ]

Description: field-argument identifies a file to include when creating a table of contents, a table of authorities, or an index using a TOC (ยง2.16.5.75), TOA (ยง2.16.5.74), or INDEX field (ยง2.16.5.35). RD fields that reference a series of files must be in the same order as the files in the final document. If the location includes a long file name containing spaces, field-argument shall contain delimiting quotes. A single backslash in the file path shall be preceded directly by a backslash.

For a complex field implementation in XML the optional field-value storage is not needed.

Field Value: None.

Switches: One of the following field-specific-switches:

\p

Indicates that the path is relative to the current document.

[Example: The following fields inserted into one document create a table of contents that includes entries from the three referenced documents:

{ TOC }
{ RD C:\\Manual\\Chapters\\Chapter1.doc }
{ RD C:\\Manual\\Chapters\\Chapter2.doc }
{ RD C:\\Manual\\Chapters\\Chapter3.doc }

end example]
